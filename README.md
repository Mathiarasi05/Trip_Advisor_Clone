# Trip_Advisor_Clone
A responsive TripAdvisor clone built with modern front-end technologies, featuring a clean UI and adaptive layout for seamless browsing across desktop, tablet, and mobile devices.

TripAdvisor Clone (Front-End)

Description

A responsive front-end clone of the TripAdvisor website built using HTML and CSS. The project focuses on recreating a real-world travel website UI with proper layout structure, spacing, alignment, and responsiveness across devices.


---

Technologies Used

HTML5

CSS3

Flexbox

Media Queries


---

Resources Used

Google Fonts

Font Awesome



---

Sections Included

The website consists of the following UI sections:

Navigation bar with logo and icons

Search bar with category options

Interest-based image cards

City and destination cards

Promotional split-layout section

Advertisement container

Footer with links and information



---

Key Front-End Concepts Applied

Flexbox for alignment and layout control

Responsive design using media queries

Card-based UI design

Consistent spacing and typography

Hover effects and visual hierarchy



---

Responsive Behavior

Desktop: multi-column layouts and horizontal sections

Tablet: adjusted spacing and resized components

Mobile: vertical stacking and optimized text sizes



---

Learning Outcome

Improved understanding of responsive layouts

Better control over Flexbox positioning

Hands-on experience cloning real-world UI designs

Strengthened CSS structuring and organization
